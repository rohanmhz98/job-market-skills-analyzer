# Job Market Skills Analyzer

## Overview
This project analyzes job postings to identify the most in-demand skills across Analyst, Business Analyst, and Data Scientist roles.

## Tools Used
- Python
- Pandas
- Matplotlib
- Streamlit

## Features
- Skill analysis by job role
- Top skills across all job postings
- Percentage-based skill insights
- Interactive web app using Streamlit

## Key Insights
- SQL is the most in-demand skill across roles
- Analyst roles focus on Excel, SQL, and dashboards
- Data Scientist roles require Python and machine learning
- Business Analyst roles emphasize reporting and communication

## Files
- app.py → Streamlit application
- jobs.csv → dataset
- requirements.txt → dependencies
